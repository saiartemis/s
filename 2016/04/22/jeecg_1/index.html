jeecg系统详解_登录模块

发表于 2016-04-22   |  
  1. 输入账号，密码和验证码，点击登录按钮

    调用js中CheckUser()方法，对用户信息进行验证。
    CheckUser()代码如下：
    function checkUser(){
    if(!validForm()){
      return false;
    }
      Login();
   }
    其中validForm()方法主要验证用户名，密码，验证码输入是否完全。
    下面我们来看一下Login()方法。
    首先通过ajax调用checkurl，验证成功后调用actionurl跳转。
    checkurl：loginController.do?checkuser
    actionurl：loginController.do?login
    loginController.do?checkuser流程：
    1.判断选择的语言种类。
    2.验证随机码
    3.判断数据库是否已经初始化
    4.判断用户名密码的合法性。
    loginController.do?login流程：
    1.获取session中的用户。
    2.通过用户查询用户权限。
    3.获取页面风格。
    4.保存cookies。
